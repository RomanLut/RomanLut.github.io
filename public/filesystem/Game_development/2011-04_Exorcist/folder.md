**Exorcist**
Hidden Object Game developed by **Deep Shadows**.

**My Role:** Hidden Object Game Engine design, Scenes and Animation editor.
**Platform:** Windows
**Language:** C++


In 2011, **Deep Shadows** switched to the hidden object games genre. Before leaving the company, I developed the tools and game engine for this direction within a few months.
