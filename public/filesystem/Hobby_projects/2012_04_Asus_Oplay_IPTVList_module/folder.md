**IPTVList**

**Date:** 2012
**Platform:** Asus O!Play media player
**Language:** xTreamer RSS

Module for **Asus O!Play** media player. It turns **Asus O!Play** into **IPTV player**.

This is an IPTV playlist with a modern, user-friendly interface, easy navigation, and support for recording TV programs. The GUI is styled after the PlayOn! firmware skin.

[Discussion thread on moservices.org forums](https://moservices.org/forum/viewtopic.php?f=5&t=1535)

**Asus O!Play** is now completely outdated.
